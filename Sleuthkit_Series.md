# Description

Sleuthkit Intro / Sleuthkit Apprentice / Disk, disk, sleuth! (1 and 2)

# Solution

- for each of this challenges we need to use the `Autopsy` tool
- just load the disk image into Autopsy via the user interface and browse through the different partitions and directories on each partition
- in the end you will find the flag (usually in the root directory)
